#  好用算法 Trick

## 很大的数

表示一个很大的数，看类型包含多少字节写多少个 `3f`。

:::info

$(3f)_{16}=(0111\ 1111)_2$

:::

```cpp
const int N = 0x3f3f3f3f;
```

## 建图

### 邻接表

```cpp
const int N = 1e5 + 10, M = 2e5 + 10;
int h[N], e[M], ne[M], idx;

void add (int a, int b) {
    e[idx] = b;
    ne[idx] = h[a];
    h[a] = idx ++;
}   
```

### 邻接矩阵

```cpp
int g[N][N];

for (int i = 0; i < m; i++) {
    g[a][b] = g[b][a] = min(g[a][b], c); // 无向图
    g[a][b] = min(g[a][b], c); // 有向图
}
```
