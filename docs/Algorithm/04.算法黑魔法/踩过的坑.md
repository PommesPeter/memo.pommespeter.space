## 精度相关

- 涉及到浮点数运算，或者浮点数转换成整型的一定要注意，计算的过程中一定要使用 float 或者 double 类型，最后如果要求要输出整数部分则直接使用类型转换即可。

  如果中间过程中不使用浮点数运算可能会因为精度导致中间结果产生较大的误差，参与计算越多，误差越大。

  ```cpp
  double sum = 0;
  // a series of computing
  cout << (int) sum << endl;
  ```
