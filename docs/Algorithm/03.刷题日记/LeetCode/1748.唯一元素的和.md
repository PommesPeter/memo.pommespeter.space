# [1748. 唯一元素的和](https://leetcode-cn.com/problems/sum-of-unique-elements/)

## 思路

用桶来存，因为数据量不大，找到只有一个数的就行。

## 代码

```cpp
class Solution {
public:
    int q[110];
    int sumOfUnique(vector<int>& nums) {
        memset(q, 0, sizeof q);
        int sum = 0;
        for (int i =0 ; i < nums.size(); i ++) {
            q[nums[i]] ++;
        }

        for (int i = 0; i < nums.size(); i++) {
            if (q[nums[i]] == 1) sum += nums[i];
        }
        return sum;
    }
};
```
